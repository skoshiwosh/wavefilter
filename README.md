This repo contains python scripts for modifying jpg images using sine wave filters. The underlying script, wave_filter.py, was generating using Claude. The wave_filter_Claude_doc.pdf displays my original query to Claude and it’s response. The script, wavewin.py is a GUI wrapper using PySide6 that also imports filterimage.py that uses json to specify arguments for function, apply_wave_filter, contained in wave_filter.py
